# Attributes

Any client can define an arbitrary set of attributes for its logs.

Some attributes are commonly defined throughout various sources.

## GCP

* gcp.resource.type

## Kubernetes

* k8s.namespace.name
* k8s.pod.name
* k8s.container.name
* k8s.pod.uid

## Http

* http.uri
* http.user\_agent
* http.verb
* http.status
* http.version

## Stripe

* stripe.event.type
* stripe.event.id
